## 🤝 Code Convention
### 커밋 메시지

```
**Commit 유형 지정**
(Create) => 새로운 컴포넌트, 기능을 완성
(Update) => 수정, 추가, 보완 등 개정
(Delete) => 삭제
(Fix) => 오타, 오류 수정

**Commit Message 구조**
type(타입) : title(제목)
WIP : 완성 전에 커밋할 경우 제목 뒤에 붙임
body(본문, 생략 가능)
```
